# Спецификация: Веб-панель администрирования Aura

**Дата:** 2026-03-23  
**Статус:** Утверждено  
**Версия:** 1.0

---

## 1. Обзор проекта

Веб-панель администрирования для системы Aura - персонализированного ухода за кожей. Панель предоставляет административный интерфейс для управления продуктами, процедурами, контентом и пользователями с разными уровнями доступа.

---

## 2. Цели и задачи

### Цели
- Создать современную веб-панель в стиле мобильного приложения
- Обеспечить разграничение прав доступа для разных ролей
- Централизованное управление контентом и продуктами

### Задачи
- Реализовать аутентификацию и авторизацию
- Разработать UI в стиле glassmorphism с цветами Mint Green/Pink/Lavender
- Создать CRUD для продуктов, процедур, контента
- Добавить дашборд с аналитикой

---

## 3. Дизайн-система

### Цветовая схвета

| Цвет | Значение | Использование |
|------|----------|---------------|
| Mint Green | `#A7F3D0` | Primary, акценты, кнопки |
| Pink | `#FB6FE8` | Secondary, важные действия |
| Lavender | `#E0C3FC` | Tertiary, категории |
| Background Light | `#F8F9FA` | Фон (light mode) |
| Background Dark | `#0A0A0A` | Фон (dark mode) |
| Surface | `#FFFFFF` | Карточки (light) |
| On Surface | `#1A1A1A` | Текст на поверхности |

### Компоненты

**Sidebar (боковая панель)**
- Ширина: 260px
- Glassmorphism эффект: `backdrop-filter: blur(10px)`
- Активный элемент: подсветка Mint Green
- Hover: легкое осветление фона

**Glass Card (стеклянная карточка)**
- Border-radius: 16px
- Фон: полупрозрачный с blur
- Тень: мягкая, цвета Mint Green
- Padding: 20px

**Кнопки**
- Primary: Mint Green фон, белый текст, тень
- Secondary: outline стиль, Mint Green граница
- Danger: Pink фон
- Border-radius: 12px

**Таблицы**
- Header: серый фон, жирный текст
- Строки: чередование светлых тонов
- Hover: легкое выделение

## 5. Разделы панели

### 5.1 Дашборд
- Статистические карточки (стеклянные)
- График активности пользователей
- Список последних действий
- Быстрые действия

### 5.2 Продукты
- Таблица продуктов с фильтрами
- Карточка продукта (фото, название, бренд, цена, совместимость)
- Модальное окно добавления/редактирования
- Поиск и сортировка

### 5.3 Процедуры
- CRUD салонных процедур
- Длительность, цена, описание
- Связь с продуктами

### 5.4 Контент (База знаний)
- Статьи с тегами
- RAG-система для AI
- Категории и поиск

### 5.5 Пользователи
- Управление пользователями
- Просмотр профилей
- История сканирований

### 5.6 Отчеты
- Экспорт данных (CSV, PDF)
- Статистика использования
- Аналитика совместимости

### 5.7 Настройки
- Управление ролями
- Настройки системы
- Профиль пользователя

## 7. API Endpoints

### Authentication
- `POST /api/auth/login` - Вход
- `POST /api/auth/logout` - Выход
- `GET /api/auth/me` - Текущий пользователь

### Products
- `GET /api/products` - Список продуктов
- `POST /api/products` - Создать продукт
- `PUT /api/products/:id` - Обновить
- `DELETE /api/products/:id` - Удалить

### Procedures
- `GET /api/procedures` - Список процедур
- `POST /api/procedures` - Создать
- `PUT /api/procedures/:id` - Обновить
- `DELETE /api/procedures/:id` - Удалить

### Content
- `GET /api/articles` - Статьи
- `POST /api/articles` - Создать статью
- `PUT /api/articles/:id` - Обновить
- `DELETE /api/articles/:id` - Удалить

### Users
- `GET /api/users` - Список пользователей
- `GET /api/users/:id` - Профиль пользователя
- `PUT /api/users/:id` - Обновить
- `DELETE /api/users/:id` - Удалить

### Reports
- `GET /api/reports/activity` - Активность
- `GET /api/reports/compatibility` - Совместимость
- `GET /api/reports/export` - Экспорт
