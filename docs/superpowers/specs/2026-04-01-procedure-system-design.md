# Система процедур косметологии - спецификация

## Обзор проекта

Создание системы управления косметологическими процедурами с детальной информацией о каждой процедуре, структурированной через wizard-интерфейс.

## Структура данных

### Направления (3, фиксировано)
- Аппаратная косметология
- Инъекционная косметология
- Эстетическая косметология

### Справочники (гибрид: базовый набор + возможность добавления)

| Справочник | Тип | Примеры значений |
|------------|-----|------------------|
| Тип методики | Список | Лазер, УЗ-терапия, RF-лифтинг, Микротоки, Криотерапия, Мезотерапия, Биоревитализация, Ботокс и т.д. |
| Продолжительность | Список | 30 мин, 45 мин, 1 час, 1.5 часа, 2 часа |
| Оборудование/препарат | Динамический | По мере добавления процедур |
| Зоны | Мульти-селект | Лицо, Шея, Декольте, Руки, Тело, Веки, Губы и т.д. |
| Желаемый эффект | Мульти-селект | Омоложение, Лифтинг, Увлажнение, Осветление, Липолиз и т.д. |
| Решаемые проблемы | Мульти-селект | Морщины, Акне, Пигментация, Целлюлит, Растяжки и т.д. |

### Поля процедуры

| Тип | Поля |
|-----|------|
| Обязательные | Название, Направление |
| Выпадающий список | Тип методики, Продолжительность, Оборудование/препарат |
| Мульти-селект | Желаемый эффект, Зоны, Решаемые проблемы |
| Текстовые (опциональные) | Описание, О процедуре, Преимущества, Показания, Принцип действия, Как проходит, Для кого подходит, Проблемы которые решает, Противопоказания, Подготовка к процедуре, Рекомендованный курс, Реабилитация, Постпроцедурный уход, Побочные эффекты |
| Медиа | Фото (несколько) |

## Пользовательский интерфейс

### Wizard (7 шагов)

1. **Базовая информация**
   - Название (required)
   - Направление (required, select)
   - Тип методики (select)

2. **Описание**
   - Описание
   - О процедуре
   - Принцип действия
   - Как проходит процедура

3. **Показания**
   - Показания к процедуре
   - Для кого подходит
   - Решаемые проблемы (мульти-селект)

4. **Эффект**
   - Желаемый эффект (мульти-селект)
   - Преимущества

5. **Подготовка**
   - Подготовка к процедуре
   - Рекомендованный курс процедур

6. **Реабилитация**
   - Реабилитация после процедуры
   - Постпроцедурный уход
   - Побочные эффекты и осложнения
   - Противопоказания

7. **Оборудование и фото**
   - Оборудование или препарат (select)
   - Зоны (мульти-селект)
   - Фото (загрузка)

### Отображение на карточке
- Пустые текстовые поля не показываются
- Заполненные отображаются компактно
- Фото в галерее

## Техническая реализация

### Backend
- Расширить модель Procedure в БД
- Добавить поля в Pydantic models
- API endpoints для CRUD

### Frontend
- Новый компонент ProcedureWizard (7 шагов)
- Обновить страницу Procedures
- Использовать существующие Select с мульти-селект
- Загрузка фото по аналогии с Products

## Критерии успеха
- Wizard позволяет ввести все данные за 7 шагов
- Справочники поддерживают добавление новых значений
- На карточке отображаются только заполненные поля
- Загрузка фото работает корректно
